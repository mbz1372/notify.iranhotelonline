# IranHotel Notify V1 Pro

نسخه V1 حرفه‌ای و سبک برای Vercel، بدون دیتابیس.

## امکانات
- داشبورد خلاصه
- ارسال پیامک با SMS.ir از طریق Serverless Function
- انتخاب هتل، انتخاب شماره‌های استخراج‌شده، ورود دستی شماره
- قالب‌های هوشمند با متغیرها
- مدیریت قالب‌ها از داخل پنل
- مدیریت هتل‌ها در مرورگر: افزودن، ویرایش، حذف، خروجی JSON
- گزارش ارسال و خروجی CSV
- واتساپ/تلگرام/بله با Deep Link و کپی خودکار متن

## Environment Variables در Vercel
در Settings > Environment Variables این دو مورد را برای Production اضافه کنید:

```env
SMSIR_API_KEY=your_smsir_api_key
SMSIR_LINE_NUMBER=3000400705
```

بعد از ثبت، Redeploy بزنید.

## نکته
قالب‌ها، هتل‌های ویرایش‌شده و لاگ‌ها در LocalStorage مرورگر ذخیره می‌شوند. برای تیمی‌کردن کامل، نسخه بعدی باید بک‌اند ذخیره‌سازی داشته باشد.
